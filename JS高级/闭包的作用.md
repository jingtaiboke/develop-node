使用函数内部的变量在函数执行完后,仍然存活在内存中(延长了局部变量的生命周期)

让函数外部可以操作(读写)到函数内部的数据(变量/函数)

<strong>缺点</strong>

函数执行完后,函数内的局部变量没有释放,占用内存时间变长

容易造成内存泄漏

<strong>解决</strong>

能不用闭包就不用

及时释放 

```
 function fn1() {
        var a = 2;
        function fn2() {
          a++;
          console.log(a);
        }
        return fn2;
      }
      var f = fn1();
      f();
      f = null;
```

让内部函数成为垃圾对象--->回收闭包
